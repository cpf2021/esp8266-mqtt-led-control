# ESP8266 MQTT LED 控制网页

这是一个用于控制ESP8266连接的WS2812智能灯带的网页应用。

## 功能特点

- 支持简单模式和高级模式控制
- 支持多种MQTT平台：巴法云、EMQX公共服务器、Home Assistant、阿里云IoT等
- 提供系统预设效果：单色、彩虹、跑马灯、流水灯等
- 支持自定义效果编辑器
- 响应式设计，适配移动端

## 技术栈

- HTML5 + CSS3
- Vue.js 2.x
- iView UI框架
- MQTT.js

## 部署说明

### 本地部署
1. 直接在浏览器中打开 `mqtt.html` 文件
2. 或使用本地服务器（如 http-server、live-server 等）

### 远程部署（GitHub Pages）
1. 克隆此仓库到GitHub
2. 在仓库设置中启用GitHub Pages
3. 选择 `main` 分支作为源
4. 访问生成的GitHub Pages URL

## 配置说明

1. 打开 `config.html` 配置MQTT连接参数
2. 选择合适的MQTT平台并填写对应参数
3. 测试连接成功后保存配置

## 使用方法

1. 在 `config.html` 中配置MQTT连接
2. 在 `mqtt.html` 中控制LED灯带
3. 在 `editor.html` 中创建自定义效果

## 注意事项

- 确保ESP8266设备已连接到相同的MQTT主题
- 对于巴法云平台，需要在巴法云官网创建设备并获取私钥
- 公共MQTT服务器可能不稳定，建议使用私有的MQTT服务器或巴法云

## 网页与设备交互指令说明

### 网页发送的指令

#### 1. 状态查询指令
- **指令格式**: `status:web`
- **作用**: 网页加载时或定期发送，用于查询设备的当前状态
- **发送时机**: 
  - 网页初始化加载时
  - 每10秒定期发送
  - MQTT连接成功后立即发送

#### 2. 开关控制指令
- **开灯指令**: `on`
- **关灯指令**: `off`
- **作用**: 控制LED灯带的开关状态

#### 3. 亮度调节指令
- **指令格式**: `brightness:<value>`
- **示例**: `brightness:128`
- **作用**: 设置LED灯带的亮度，取值范围0-255

#### 4. 预设效果指令
- **指令格式**: `preset:<type>`
- **示例**: `preset:rainbow`
- **支持的预设类型**:
  - 单色: `red`, `green`, `blue`, `yellow`, `purple`, `cyan`, `white`
  - 动态效果: `rainbow`, `running`, `flow`, `wave`, `bounce`, `flash`, `gradient`

#### 5. 自定义效果指令
- **指令格式**: `c:{"l":<loop>,"f":<frames>}`
- **示例**: `c:{"l":1,"f":[[100,0,255,0,0]]}`
- **参数说明**:
  - `l`: 是否循环播放，1为循环，0为单次
  - `f`: 帧数据数组，每个帧包含延迟时间和LED颜色数据

### 设备响应指令

#### 1. 状态回复
- **响应格式**: `status:{"status":<status>,"preset":<preset>}`
- **示例**: `status:{"status":1,"preset":"rainbow"}`
- **参数说明**:
  - `status`: 设备状态，1为开启，0为关闭
  - `preset`: 当前运行的预设效果名称

#### 2. 状态变更通知
- **设备开启**: `on`
- **设备关闭**: `off`
- **作用**: 当设备状态发生变化时，主动通知网页

### 巴法云平台特殊处理

对于巴法云平台，指令格式会进行如下转换：

- **开灯**: `on` → `on`
- **关灯**: `off` → `off`
- **亮度调节**: `brightness:128` → `br#128`
- **颜色设置**: 转换为 `color#r,g,b` 格式

### 设备离线检测机制

- 网页会记录设备的最后活动时间
- 如果30秒内没有收到设备的任何消息，网页会将设备标记为离线
- 设备离线时，所有控制按钮会显示为禁用状态

### 预设按钮高亮机制

- 当设备回复状态时，网页会根据回复的 `preset` 值高亮对应的预设按钮
- 如果设备运行的是自定义效果，预设按钮不会高亮
- 如果设备关闭，预设按钮也不会高亮
